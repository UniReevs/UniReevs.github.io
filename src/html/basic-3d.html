立体図形の回転

立方体・直方体をいろんな角度から描けるようになったら、他の形も描けるようになる。色んなアングルの一覧はこちら

矢印の上下で1度、左右で10度、角度が変わる。

Options
height
360
replay

Cube

Cuboid

練習

  1. ひたすら眺めてどの方向へどう変形するのか観察
  2. ある程度わかったら、x, y, zそれぞれの方向へ角度を変えた時の形を順番に描く
  3. 箱型がわかったら、箱をシリンダー・ピラミッド・球に変えて同じように練習
  4. 目に入るありとあらゆる物体をとりあえず箱型に置き換えたらどう感じになるか脳内でシミュレーションする
  5. いろんな基本3Dシェイプを紙の上にバランスよく配置して描く練習をひたすらする
  • smart_display箱を回転させる時の考え方
