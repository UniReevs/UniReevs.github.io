Basic 2D Shapes (平面の基本の形)

どういう練習をしたか

コピー用紙で練習するのがおすすめ。理由は描くときに紙をくる回すのが楽だから。腕や指の動きの関係上、線を引きやすい方向があって、その方向に向かって描くと時間が短縮できるし、綺麗に描ける。

練習する時は、ドットや三角の図形を基準にどのくらいの角度や距離感で線を引くのかと言うのを考えながら練習した。こういう練習をすると後々より複雑な形を描くときに役立つだろうと思っている。

そして練習する時は正確な円や正三角形の図形を参照しながら練習した。理由は何も参照するものがない状態で練習してバランスの崩れた形を覚えたくなかったから。

プリント用

  • A4 ドット方眼紙
  • A4 ドット方眼紙 + 参照のための正三角形・正方形・楕円付き

  • A4 等角図紙 + 参照のための円・楕円付き
